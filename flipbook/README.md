# School Newspaper Website

### Objective

Help schools build and publish student run newspapers

### Modules

#### Article composer

- Students submit articles as google sheets documents
- Convert Google sheets to markdown add frontmatter

#### PDF Generator

- Use Jekyll blogging software to convert Markdown articles as PDF
- Use frontmatter and priority to organize the layout
- Generate PDF from Blog Web content

#### Publish PDF as Flipbook

- Use Python frameworks to store and manage PDF uploads and moderation
- Use Python to rasterize PDF pages into individual images
- Use StPageFlip to render these images with a curl effect to make a flipbook feel
